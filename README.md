# MemoryGameJsProject
  Basic Memory Game 

# Project Goals
  The game randomly shuffles cards,
  Matches Cards,
  Gives Total Moves,
  Shows a Star Rating,
  Has a timer: stop/start,
  Restart button,
  Gives a Congratulations Popup

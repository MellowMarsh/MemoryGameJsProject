# MemoryGameJsProject
  Basic Memory Game built from scratch. I used what I learned and incorporated my skills into a FEND project.

# Project Goals
  The game randomly shuffles cards,
  Matches Cards,
  Gives Total Moves,
  Shows a Star Rating,
  Has a timer: stop/start,
  Restart button,
  Gives a Congratulations Popup
